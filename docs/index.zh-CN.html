localmelo 文档

localmelo 是一个 pre-alpha、本地优先的 agent runtime，核心能力是显式 memory 分层、受检查的 tool execution，以及连接用户自己管理的本地或云端模型 endpoint。

online loop 可验证 memory 进行中 sleep 后续计划

安装配置

先安装 package，配置 chat backend，然后运行一次 direct query。这是穿过当前 runtime 的最小可用路径。

minimum local run
pip install -e ".[dev,gateway]"
melo --reconfigure
melo "What is 6*7?"
打开快速开始 查看架构

localmelo 提供什么

Direct CLI 从命令行执行一次 agent query，然后退出。 Gateway API 用 HTTP 暴露同一个 runtime，并支持 session 复用。 受检查的工具执行 校验 ingress、planning、execution、output 和 memory write。 分层 Memory 把 working memory、history、long-term retrieval 和 tools 拆开维护。 后端连接器 连接 Ollama、MLC、vLLM、SGLang 或支持的云端 provider。 路线图跟踪 跟踪 runtime、memory、backend 和 release quality 的进展。
